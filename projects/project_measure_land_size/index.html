Measure Land and House Size

An web application to measure size of land and house

Introduction

I have tried to search around for my first house in Melbourne, Australia. The most comprehensive platform to do so is RealEstate.com.au. However, I found that not all the properties have the land size information. More importantly is the house size which can give me an idea of how big the house is, which is even less available. To get this kind of information, I have to go to Google Map and measure it manually with the measuring tool. This is a tedious task, especially when I have to do it for many properties. I believe that there are many buyers who are facing the same problem like me. And an automated tool to measure the land and house size will be very helpful for them.

Competitors

Landchecker

Landchecker is a tool to check the land size and planning information of a property. It is a paid service, and the subscription plan is not for individual users (i.e., the Standard plan is $75 per month but with 10 properties report per month, and if you want to unlimited reports, you have to pay $240 per month for the Advanced plan).

The information provided by Landchecker is very comprehensive. It includes the land size, planning zones, planning overlays, planning permits, planning applications, planning history, planning controls, planning scheme, and planning maps.
